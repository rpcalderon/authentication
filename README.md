# Authentication

Node Authentication Starter

## Getting Started

```sh
cd authentication
npm install
node index.js
```

---

## Documentation

[Node Authentication API Documentation](nodeauthentication.docs.apiary.io)

Use with Node version 8.11.3

- [NodeJS](http://www.nodejs.org/) - The web framework used
